frab by oneiros

conference manangement system

frab is a web-based conference planning and management system. It helps to collect submissions, to manage talks and speakers and to create a schedule. frab is based upon Ruby on Rails . It was created to organize FrOSCon , where it replaced pentabarf as the primary conference planning tool.

Install

frab is hosted at github . There is no official release yet. Just clone the repository and get the latest code. Please refer to the README for installation instructions.
